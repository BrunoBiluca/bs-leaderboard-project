# Api Leader Board

## Technologies
EFCore 3
EntityFrameworkCore 3.1.2
SQLite

## Compilation/Execution
- Dependencies
```sh
dotnet tool install --global dotnet-ef
```

- Compilation
```sh
dotnet restore .\APIASPNETCoreCQRSEFCore.sln
dotnet build .\APIASPNETCoreCQRSEFCore.sln
```

- Execution
```sh
cd .\GTSharp.Domain.Api\
dotnet watch run 
```
